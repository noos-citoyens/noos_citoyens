Guide de rédaction des propositions

Nous vous exposons ici quelques conseils de rédaction des propositions afin d’en faciliter la compréhension, les traitements, les analyses, et la synthèse. Afin de rester dans la légalité il vous est demandé de proscrire toute proposition dont la diffusion contreviendrait à la loi : Pas d’insultes, pas de menaces, pas de violence verbale… Respectons la loi, nous souhaitons inscrire cette démarche dans un cadre constructif et transparent. Les propositions à caractère commerciales et les liens publicitaires sont aussi à proscrire.

Toutes les propositions déposées par les citoyens sur le site NOos-citoyens seront traités et analysées avec des outils de traitement automatique des langues naturelles. Cela permettra par exemple de regrouper les propositions semblables ou bien encore de rechercher les propositions sur tel ou tel sujet avec un moteur de recherche. D’autre part, nous avons limité les propositions sous la forme d’un texte en français de 280 caractères, il vous est donc conseillé d’être concis et claire avec des propositions simples.

Nous proposons de raisonner en « Problème » / « Solution »
Autrement dit : « Quel est le problème que je veux soulever ?» / « quelle solution je propose pour le résoudre ».

Par exemple :
Le problème : « Le smic est trop bas »
La solution : « Augmenter le SMIC de 20% »

Vous pouvez proposer autant de propositions que vous le souhaitez. Pour un même problème vous pouvez proposer plusieurs solutions. Pensez à séparer les propositions en les inscrivant en plusieurs temps.


Par exemple :
Le problème : « Le pouvoir d’achat qui diminue ».
La solution : « Indexer les salaires sur l’inflation »

Le problème : « Le pouvoir d’achat qui diminue ».
La solution : « Indexer les retraites sur l’inflation »

Le problème : « Le pouvoir d’achat qui diminue ».
La solution : « Supprimer les taxes sur les produits de première nécessité ».
Bonne rédaction !
